# 异环驱动计算器 — 繁體中文補丁

讓開源工具 异环驱动计算器（簡中版）能正確辨識繁體中文版《異環》的裝備詞條，
用於鑑定 / 配裝驅動與卡帶。

> ⚠️ 執行時會自動從原作者官方庫下載原程式到你的電腦。
> 本補丁本身不含原程式碼，只是「疊加」在其上做簡轉繁與辨識修正。

---

## 這是什麼

- 原程式：[hxwd94666/NTE-Drive-Calc](https://github.com/hxwd94666/NTE-Drive-Calc)（作者：hxwd94666）
- 本補丁做的事：
  1. 用 OpenCC 把原程式的設定檔與程式碼簡轉繁（繁體 UI）
  2. 在 OCR 辨識後做正規化（簡轉繁 + 修常見誤字），讓辨識結果對得上繁體詞條
  3. 修正原程式在原始碼執行時的相容性問題（管理員提權、多螢幕 DPI 等）
  4. 提供一個圖形化「控制台」，統一處理安裝、更新、啟動、修復與版本檢查

本補丁僅包含作者以外、由本專案自行撰寫的檔案，不含原程式碼。

---

## 怎麼用

1. 下載本補丁（Code → Download ZIP），解壓到任一資料夾
2. 雙擊 `異環計算器繁中控制台.bat`
打開後是一個圖形控制台，剩下的都在裡面點：

| 按鈕 | 作用 |
|------|------|
| 🚀 啟動程式 | 開始使用 |
| ⬇ 更新主程式 | 第一次安裝 / 之後更新都按這個（會自動從官方下載原程式並轉繁） |
| 🔧 修復 | 套件 / 驅動有問題時按 |
| 🔄 重新檢查 | 重看環境與版本狀態 |
| 📖 使用說明 | 開啟說明 |

- 控制台上方會顯示環境是否正常（Python / 套件 / 驅動 / 已轉繁）與是否為最新版，綠燈代表 OK。
- 沒有 Python 也沒關係，控制台會自動幫你安裝。
- 第一次：開控制台 → 按「更新主程式」→ 按「啟動程式」。

詳細說明見 `使用說明.txt`。
